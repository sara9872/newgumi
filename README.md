# LocalHub 구미/경북

공공데이터 기반 구미/경북 지역 정보 공유 커뮤니티입니다. Vue 3 + Vite 정적 SPA로 구성되어 있으며, 제공 JSON 데이터를 프론트엔드에서 직접 불러옵니다.

## 주요 기능

- 홈: 구미/경북 소개, 추천 관광지/음식점, 축제 일정, 최근 게시글
- 지역 정보: 관광지, 음식점, 숙박, 문화시설, 쇼핑, 레포츠, 여행코스 목록과 검색
- 익명 게시판: 목록, 상세, 작성, 수정, 삭제
- 게시판 추가 기능: 검색, 좋아요, 북마크
- 축제 캘린더: 축제 시작일/종료일 기반 월별 달력
- AI 챗봇: OpenAI API와 제공 JSON 데이터를 활용한 지역 안내

## 실행 방법

```bash
pnpm install
pnpm dev
```

## 빌드

```bash
pnpm build
```

## 환경 변수

`.env.example`을 참고해 `.env` 파일을 만들고 제한된 사용량의 API 키를 넣습니다.

```env
VITE_OPENAI_API_KEY=your_limited_openai_api_key
```

Vite 환경 변수는 브라우저 번들에 포함되므로 반드시 사용량 제한이 걸린 키만 사용해야 합니다.

## 데이터 출처

- 제공 기관: 한국관광공사
- 데이터: TourAPI 4.0
- 공공데이터 포털: https://www.data.go.kr/data/15101578/openapi.do
- 라이선스: 공공누리 제3유형, 출처 표시 및 변경 금지

## 제출 전 확인

- `.env`가 Git에 포함되지 않았는지 확인
- Netlify 배포 URL에서 새로고침 라우팅이 정상인지 확인
- 게시글은 localStorage에 저장되므로 다른 브라우저와 공유되지 않는다는 점을 발표에서 설명
