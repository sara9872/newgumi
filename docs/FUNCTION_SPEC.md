# LocalHub 기능 명세서

## 서비스 개요

LocalHub는 구미/경북권 공공데이터를 활용해 지역 주민과 관광객이 정보를 확인하고 익명으로 게시글을 작성할 수 있는 정적 SPA 서비스입니다.

## 필수 기능

| 기능 | 구현 내용 |
| --- | --- |
| Vue 3 SPA | Vue 3, Vite, vue-router 기반 화면 구성 |
| 제공 JSON 연동 | `public/data`의 구미/경북 JSON 파일을 fetch로 직접 로드 |
| 익명 게시판 | 회원가입 없이 게시글 목록, 상세, 작성, 수정, 삭제 구현 |
| localStorage 저장 | `localhub_posts`, `localhub_bookmarks` 키 사용 |
| 비밀번호 확인 | 수정/삭제 시 작성자가 입력한 비밀번호와 비교 |
| 챗봇 | OpenAI API를 직접 호출하고 지역 데이터 요약 컨텍스트 사용 |
| 배포 대응 | Netlify SPA redirect 설정 포함 |

## 선택 기능

| 기능 | 구현 내용 |
| --- | --- |
| 축제 캘린더 | 축제공연행사 JSON의 `eventstartdate`, `eventenddate`를 달력으로 표시 |
| 게시판 추가 기능 | 검색, 좋아요, 북마크 구현 |

## 화면 구성

- 홈: 서비스 소개, 추천 관광지/음식점, 축제 일정, 최근 게시글
- 지역 정보: 카테고리 탭, 검색, 카드 목록
- 게시판 목록: 검색, 북마크 필터, 글쓰기
- 게시판 상세: 조회수, 좋아요, 북마크, 수정 이동, 삭제
- 게시글 작성/수정: 제목, 내용, 비밀번호 입력
- 축제 캘린더: 월 이동, 날짜별 축제 표시, 상세 패널
- 챗봇: 플로팅 버튼, 대화창, 예시 질문, API 실패 fallback

## 데이터 출처 및 라이선스

- 제공 기관: 한국관광공사
- 데이터명: 국문 관광정보 서비스 TourAPI 4.0
- URL: https://www.data.go.kr/data/15101578/openapi.do
- 권역: 구미/경북권
- 라이선스: 공공누리 제3유형
- 조건: 출처 표시, 변경 금지

## 한계

- 별도 서버가 없으므로 게시글은 작성한 브라우저의 localStorage에만 저장됩니다.
- OpenAI API 키는 프론트엔드 번들에 노출될 수 있으므로 사용량 제한 키만 사용해야 합니다.
- 날씨 API와 실시간 알림은 일정상 후순위 기능으로 제외했습니다.
