# 발표 자료 구성안

## 1. 서비스 소개

- LocalHub 구미/경북
- 공공데이터 기반 지역 정보 공유 커뮤니티
- 대상: 지역 주민, 관광객, 축제 방문 예정자

## 2. 사용 데이터

- 한국관광공사 TourAPI 4.0
- 구미/경북권 관광지, 음식점, 축제공연행사, 숙박, 문화시설, 쇼핑, 레포츠, 여행코스
- 공공누리 제3유형: 출처 표시, 변경 금지

## 3. 핵심 기능

- 지역 정보 카테고리 조회와 검색
- 익명 게시판 CRUD
- 수정/삭제용 비밀번호 확인
- OpenAI 기반 지역 안내 챗봇

## 4. 선택 기능

- 축제 캘린더
- 게시판 검색, 좋아요, 북마크

## 5. 시연 흐름

1. 홈에서 추천 지역 정보 확인
2. 지역 정보 화면에서 음식점 또는 관광지 검색
3. 게시판 글 작성
4. 작성한 글 상세에서 좋아요, 북마크 확인
5. 비밀번호로 수정 또는 삭제 확인
6. 축제 캘린더에서 축제 상세 확인
7. 챗봇에 “구미 축제 추천해줘” 질문

## 6. 한계와 개선 방향

- localStorage 기반이라 다른 브라우저와 게시글 공유 불가
- API 키 노출 위험 때문에 제한 키 필요
- 추후 날씨 API, 지도 시각화, 실시간 알림 확장 가능
